# Secure Access Control System (Flask Demo)

## Run locally

1. Create and activate a virtual environment:

   ```bash
   python -m venv .venv
   source .venv/bin/activate
   ```

2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```

3. Start the app:

   ```bash
   python app.py
   ```

4. Open your browser:

   ```
   http://127.0.0.1:5000
   ```

The app stores users in `users.db` at the project root.
